Инструкции
    • Участники исследования объединяются в пары: один из участников становится Участником А, а второй - участником Б.
    • Каждый из участников получает по {{ Constants.tg_endowment|c }}.
    • Участник А может послать участнику Б любую сумму от 0 до {{ Constants.tg_endowment_cents }} центов (от 0$ до {{ Constants.tg_endowment|c }}).
    • Cумма, отправленная участником А, будет умножена на {{ Constants.tg_coef }}. Например, если участник А посылает участнику Б 50 центов, то участник Б получает 50*3=150 центов ($1,50).
    • В свою очередь, Участник Б может вернуть назад участнику А любое число из полученной суммы.
      • {% if subsession.solo %}
    • Пример 1: Участник А решил послать участнику Б 50 центов. Участник Б получил 50*3=150 центов и 75 из них послал назад участнику А. Участник А получит бонус 100-50+75 = 125 центов ($1,25). Участник Б получит: 100+150-75 = 175 центов ($1,75).
    • Пример 2: Участник А решил ничего не посылать участнику Б. Участник А получит бонус 100 центов ($1). Участник Б получит: 100 центов ($1.0).
      • {% else %}
    • Участник Б может быть из любого из трех регионов России, которые принимают участие в исследовании.
    • Поэтому участнику А необходимо будет принять три решения - по одному для каждого региона. Одно из этих решений определит бонус участника А: в зависимости от того, откуда будет участник Б.
    • Пример 1: Участник А решил послать участнику Б из региона X 50 центов. Участник Б получил 50*3=150 центов и 75 из них послал назад участнику А. Участник А получит бонус 100-50+75 = 125 центов ($1,25). Участник Б получит: 100+150-75 = 175 центов ($1,75).
    • Пример 2: Участник А решил ничего не посылать участнику Б из региона X. Участник А получит бонус 100 центов ($1,00). Участник Б получит: 100 центов ($1,00).
      • {% endif %}
